# Portfolio

Production-ready personal portfolio and internship project built with Next.js, TypeScript, Tailwind CSS, Supabase, Prisma, and Resend.

## Status

**Phase 1 — Static foundation (in progress)**

Project scaffolding, design tokens, theme support, UI primitives, layout shell, and placeholder routes are complete. Page content comes next.

## Stack

- Next.js (App Router) + React + TypeScript
- Tailwind CSS + ESLint + Prettier
- Framer Motion + Lucide React + next-themes
- class-variance-authority + clsx + tailwind-merge
- React Hook Form + Zod (wired in Phase 2)
- Prisma + Supabase (wired in Phase 2/3)
- Resend (wired in Phase 2)
- Vercel (deployment)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Copy `.env.example` to `.env.local` before Phase 2 backend work.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Development server |
| `npm run build` | Production build |
| `npm run start` | Start production server |
| `npm run lint` | ESLint |
| `npm run format` | Prettier write |
| `npm run format:check` | Prettier check |
